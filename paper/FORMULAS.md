# Formula spec: GARCH(1,1) simulation, estimation, forecasting, and losses

Notation: `r_t` return, `sigma_t^2` conditional variance, `z_t` iid N(0,1)
innovation, `omega, alpha, beta` GARCH parameters, `p = alpha + beta`
persistence, `lrv` long-run (unconditional) variance, `T` sample length.
All quantities are per observation (one bar). The DGP is synthetic and seeded,
so `sigma_t^2` is KNOWN ground truth.

---

## 1. GARCH(1,1) data-generating process

**Source:** Bollerslev (1986) [`bollerslev1986garch`], generalizing the ARCH
model of Engle (1982) [`engle1982arch`]. Zero conditional mean:

```
r_t      = sigma_t * z_t,           z_t ~ iid N(0, 1)
sigma_t^2 = omega + alpha * r_{t-1}^2 + beta * sigma_{t-1}^2
```

Constraints: `omega > 0`, `alpha, beta >= 0`, and `alpha + beta < 1` for
covariance stationarity. The simulator seeds `sigma_0^2 = lrv` and discards a
burn-in so the retained path starts from the stationary distribution.
`sigma_t^2` (the exact conditional variance used to draw `r_t`) is returned as
ground truth.

Base ("crypto-like") DGP: `omega = 0.04`, `alpha = 0.09`, `beta = 0.90`, so
`p = 0.99` (near-IGARCH) and `lrv = 4.0` (per-observation volatility 2.0).

## 2. Long-run variance, persistence, half-life

Taking unconditional expectations of the recursion:

```
lrv = omega / (1 - alpha - beta)                       (exists iff p < 1)
persistence = alpha + beta
half_life = ln(0.5) / ln(alpha + beta)                 (in observations)
```

The half-life is the number of steps for a variance shock to decay halfway
back to `lrv`. Sensitivity of the long-run variance to persistence,
`d(lrv)/dp = lrv / (1 - p)`, diverges as `p -> 1` (near-IGARCH fragility).

## 3. Estimation: Gaussian maximum likelihood

**Source:** conditional-normal likelihood, standard for GARCH. With
`r_t | F_{t-1} ~ N(0, sigma_t^2(theta))`, the log-likelihood is a sum of
one-step-ahead densities:

```
loglik(theta) = -0.5 * sum_t [ ln(2*pi) + ln sigma_t^2(theta)
                               + r_t^2 / sigma_t^2(theta) ]
```

Maximized numerically. We use the `arch` library (Sheppard,
[`sheppard2023arch`]): `arch_model(r, mean="Zero", vol="Garch", p=1, q=1,
dist="normal", rescale=False)`. The DGP is scaled so returns are O(1), so no
`x100` rescaling is needed (`rescale=False`). Note the `arch` convention: its
`p` is the number of lagged variances (beta terms) and `q` the number of
lagged squared residuals (alpha terms).

## 4. One-step-ahead conditional-variance forecasters

All are causal (use only information up to `t-1`).

- **GARCH filter** (correctly specified), fixed fitted params:
  `h_t = omega_hat + alpha_hat * r_{t-1}^2 + beta_hat * h_{t-1}`.
  With the TRUE parameters this reproduces `sigma_t^2` exactly after the seed
  transient.
- **EWMA / RiskMetrics** [`riskmetrics1996`], decay `lambda = 0.94`:
  `h_t = (1 - lambda) * r_{t-1}^2 + lambda * h_{t-1}`. This is the constrained
  GARCH special case `omega = 0`, `alpha = 1 - lambda`, `beta = lambda`
  (persistence exactly 1).
- **Rolling-window sample variance**, windows `w in {22, 66, 132}`:
  `h_t = mean(r_{t-w..t-1}^2)`.

## 5. Analytic multi-step GARCH forecast (mean-reversion)

**Source:** iterating the recursion in conditional expectation, using
`E_T[r_{T+k}^2] = E_T[sigma_{T+k}^2]`:

```
E_T[sigma_{T+h}^2] = lrv + (alpha + beta)^{h-1} * (sigma_{T+1}^2 - lrv)
```

The forecast term structure starts at the one-step value `sigma_{T+1}^2` and
decays geometrically toward `lrv` at rate `p^{h-1}`. In the near-IGARCH regime
(`p ~ 0.99`) the decay is very slow (e.g. `0.99^{21} ~ 0.81` at `h = 22`).

## 6. Forecast-loss functions (imperfect-proxy robust)

**Source:** Patton (2011) [`patton2011volatility`]. For a variance forecast
`h` scored against a target `X` (the true variance `sigma^2`, or a
conditionally unbiased proxy such as the squared return `r^2`, since
`E[r^2 | F_{t-1}] = sigma^2`):

```
MSE(X, h)   = mean( (X - h)^2 )                           (robust)
QLIKE(X, h) = mean( X/h - ln(X/h) - 1 )                   (robust)
MAE(X, h)   = mean( |X - h| )                             (NOT robust)
```

QLIKE is the Bregman divergence generated by `-ln`; it is non-negative,
uniquely minimized at `h = X`, and depends only on the ratio `X/h` (scale
free). Patton's robustness result: the ranking of forecasters by expected MSE
or QLIKE is identical whether `X` is the latent variance or a conditionally
unbiased noisy proxy. MAE targets the conditional median of the proxy rather
than its mean, so its ranking is not proxy-robust in general.

## Verification notes

- Sections 1--5 are textbook GARCH results (Bollerslev 1986; Engle 1982); the
  half-life and mean-reversion identities are derived directly from the
  recursion and reproduced in the accompanying blog draft.
- Section 6's robust-loss characterization is Patton (2011), Prop. 1--2; MSE
  and QLIKE are his leading examples of robust losses, MAE a standard
  non-robust counterexample. Hansen & Lunde (2005)
  [`hansenlunde2005garch`] is the canonical GARCH(1,1) forecast-comparison
  study; Andersen & Bollerslev (1998) [`andersenbollerslev1998answering`] is
  the origin of the "squared returns are a noisy but unbiased proxy" framing.
- All numeric results are produced by `scripts/run_all.py` from fixed seeds and
  checked against `results/results.json` by `scripts/check_paper_numbers.py`.
